# Labb4API

1. Get alla personer - https://localhost:7027/api/Person

2. Hämta alla intressen kopplade till en person via personid i kopplingstabllen Link - https://localhost:7027/api/Interest/4

3. Hämta alla länkar kopplade till en person via personid - https://localhost:7027/api/Link/4

4. kopplar en ny person genom att göra URL nullable eftersom Link är min kopplingstabell och lägger in kopplingen där.

5. Fyller i alla uppgifter i kopplingstabllen och då skapar jag en ny länk och lägger till ett intresse.
